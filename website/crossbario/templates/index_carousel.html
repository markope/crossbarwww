The Basics

One Connection for Everything

Crossbar.io is a router for the open WAMP protocol. WAMP client implementations are available across 9 languages, enabling you to connect anything to Crossbar.io. You get both efficient data distribution using PubSub and a dedicated Command & Control channel, so that all application messaging can be via a single protocol

Remote Procedure Calls

With the Remote Procedure Call pattern, any component can register a procedure that any other component can call. Crossbar.io routes calls to the component that registered the respective procedure and returns the result to the caller.

Publish & Subscribe

Publish and Subscribe allows for the simple, scalable distribution of information. A component publishes data on a topic to the router, and this dispatches events to all components which are presently subscribed to the topic.

Multi-language

WAMP support is available for 9 languages (and growing). Implementations are open source and under liberal licenses, so you can integrate them into your free and commercial projects.

Polyglot Applications

You can mix components written in different languages within the same application. Choose the right tool for the job at a component level instead of for your entire project. Integrate third-party components or even across applications without caring about implementation language.

Real-time connections

Connections are persistent, bi-directional and low latency. For the roundtrip between our office in Erlangen and cloud hoster in Frankfurt, WAMP communication is ~2ms slower than ping. The default transport is WebSocket, enabling native connections into the browser.
